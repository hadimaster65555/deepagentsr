# Read a JSON Lines event trace

Read a JSON Lines event trace

## Usage

``` r
read_events_jsonl(path)
```

## Arguments

- path:

  JSONL path produced by
  [`export_events_jsonl()`](https://hadimaster65555.github.io/rdeepagent/reference/export_events_jsonl.md).

## Value

A list of event records.
