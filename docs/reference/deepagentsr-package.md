# deepagentsr: Deep-Agents-Style Agent Harness for R

`deepagentsr` provides a compact agent harness with R tools, virtual
filesystems, subagents, skills, memory, permissions, human approval, and
event traces. The package is designed to use `ellmer` chat objects as an
optional LLM substrate while keeping deterministic fake-model tests
local.
