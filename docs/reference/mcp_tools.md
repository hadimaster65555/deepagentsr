# Return optional MCP-backed tools

Return optional MCP-backed tools

## Usage

``` r
mcp_tools(config = NULL, tools = list(), ...)
```

## Arguments

- config:

  Optional mcptools MCP server configuration path.

- tools:

  Optional list of already materialized
  [`deep_tool()`](https://hadimaster65555.github.io/rdeepagent/reference/deep_tool.md)
  objects.

- ...:

  Reserved for future `{mcptools}` server configuration.

## Value

A list of tools.
